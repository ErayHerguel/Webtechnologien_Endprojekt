block and inlline elements

In short, block elements such as <p>, <div>, <section>, <img>, <li>, etc. expand to fill all the horizontal space available. This means that the element following the block level element will display below that block level element.

Block elements

  • Always expand to fit the width of their (parent) container, but will only expand vertically to sccomodate their content (an empty block element will be the width of its container, but have a height of 0)
  • Force newlines before and after.
  • Accept width, height, margins, padding, and border.
  • Wrap tightly around theit content both horizontally and vertically.
  • Flow like text (do not create line breaks).
  • Can exist within block elements.
  • Accept left and right margins, but not top or bottom.
  • Accept and border or padding.
  • Ignore set widths and heights.
