object-fit

It defindes an object container (in this example 100 x 200 px) with the following values:

  • fill: The image fills the given dimensions - even if it must be streched/shrinked
  • contain: The image keeps its aspect ratio - this means, that it probably will be resized to fit into the container
  • cover: The image keeps its aspect ratio - and probably must be clipped to fit into the container
